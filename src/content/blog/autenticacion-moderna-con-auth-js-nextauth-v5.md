## Auth.js v5: El futuro de la autenticación

La versión 5 de Auth.js trae cambios significativos, especialmente para aplicaciones que corren en el Edge runtime.

### ¿Qué hay de nuevo?

- Nueva configuración simplificada.
- Soporte nativo para Middleware.
- Mejor integración con React Server Components.

Recuerda siempre proteger tus rutas sensibles utilizando el middleware de Next.js.
